HIVE.

Click on the image to be sent to a webpage where you can move around a boat with your arrow keys. That's it. You just move around a little boat.

This was a project created with HTML, CSS, and the Javascript library ThreeJS. I also used Blender. It was exciting to create a project that allowed user interaction in a playful manner. ThreeJS is a library with incredible possiblities for web design.
